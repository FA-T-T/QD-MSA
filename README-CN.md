## 🌟 项目概览

在社交媒体和短视频时代，从多种模态（音频、文本、图像）理解情感内容至关重要。量子计算凭借其叠加和纠缠等固有特性，在概念上非常适合多模态学习，特别是模态融合。然而，当前的量子硬件限制，尤其是量子比特稀缺性，阻碍了其实际应用。

QD-MSA 是首个将量子电路分割技术应用于多模态情感分析的框架，有效地减少了量子比特的使用，从 $n$ 个减少到 $n/2 + 1$ 个，从而能够在 NISQ 设备上执行更复杂的量子程序。我们的框架还包含一个分布式工作流程，用于利用量子计算机集群，显著提升了计算性能。

通过结合经典神经网络进行特征提取和量子模型进行特征融合，QD-MSA 在实现卓越性能的同时，与经典深度学习方法相比，参数效率显著提高。

## 📝 论文信息

**题目：** QD-MSA: 面向量子分布式张量网络的多模态情感分析框架

**期刊：** Information Fusion

**摘要：**

多模态情感分析，整合了音频、文本、图像等数据类型，在社交媒体和短视频平台时代对于理解情感内容至关重要。量子计算凭借其固有的叠加和纠缠等特性，在概念上非常适合多模态学习，特别是模态融合。然而，当前的量子计算机面临限制，例如可用量子比特数量有限，阻碍了它们超越经典计算（量子霸权）的能力。在这项工作中，我们提出了 QD-MSA，一个量子分布式多模态情感分析框架，这是首个将量子电路分割技术应用于多模态情感分析的框架，将量子比特的使用从 $n$ 个减少到 $n/2 + 1$ 个。这一进展通过部分克服量子比特稀缺性，使得在噪声中等规模量子 (NISQ) 设备上执行更复杂的量子程序成为可能。此外，QD-MSA 包含一个新颖的工作流程，将我们的模型集成到量子计算机集群中，显著提高了计算性能，释放了 NISQ 时代量子计算机的潜力。通过结合经典神经网络进行特征提取和量子模型进行特征融合，我们的方法在节省量子资源的同时实现了卓越的性能。在 CMU-MOSEI 和 CMU-MOSI 数据集上的实验评估表明，我们的模型达到了与基于深度学习的方法相当或更优的性能，在关键指标上取得了显著改进。此外，我们的工作代表了量子计算原理首次成功集成到多模态情感分析中，实验证实所提出的模型显著优于仅依赖于量子启发策略的经典方法。这些贡献为多模态情感分析建立了一个可扩展且高效的框架，利用经典和量子计算范式来推动该领域的发展。

## ✨ 贡献

这项工作做出了五项主要贡献：

1.  **新颖的分布式工作流程：** 我们提出了一种新颖的分布式工作流程，将我们的量子混合模型集成到量子计算机集群中，显著提高了计算性能，并释放了 NISQ 设备在复杂多模态情感分析中的全部潜力。
2.  **开创性的量子多模态情感分析模型：** 我们引入了一种开创性的量子多模态情感分析模型，该模型采用量子编码和量子张量网络来融合不同的模态。这一贡献不仅突出了量子计算在 NISQ 时代的新应用，还强调了我们的方法在实际视频分析和人类情感分类中的重要性。
3.  **显著的参数效率：** 我们的方法以显著的参数效率实现了多种模态的有效融合，仅需要模态融合中传统基于深度学习模型使用的参数的 1%，从而提高了效率和可扩展性。
4.  **多模态情感分析中的量子电路分割：** 我们首次将量子电路分割技术应用于多模态情感分析问题，将量子比特的使用从 $n$ 个减少到 $n/2 + 1$ 个，节省了近 50% 的量子比特。这一显著的减少使得在量子比特有限的设备上执行更大的量子电路成为可能。
5.  **卓越的性能：** 在 CMU-MOSEI 和 CMU-MOSI 数据集上进行的大量实验表明了卓越的性能，准确率分别达到 81.62% 和 77.80%，优于先进的深度学习模型以及仅利用量子启发策略的经典模型。
